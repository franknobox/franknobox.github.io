点阵武装 Dot Arsenal

2D Roguelite 动作游戏，核心系统包括：

  • 5×5 点阵武器合成系统：基于锚点偏移算法的形状识别逻辑，支持图案平移匹配；引入"点位纯度检查"机制，通过 ScriptableObject 实现配方配置化管理，确保合成精度与扩展性。
  • 统合武器控制框架：WeaponController 基类通过策略模式统合 Melee（碰撞体检测）与 Ranged（投射物发射）逻辑；建立"耐久度剥落-资源回收"闭环，损坏武器按比例返还 Dot 资源。
  • 集成 Spine 的智能 AI：集成 Spine Mecanim 系统，解决 2D 骨骼动画下的发射点同步与动态翻转问题；实现具备放风筝与巡逻追击行为的状态机 AI。
  • 程序化地牢与房间管理：DungeonManager 负责随机房间序列生成（战斗/休息）与 BGM 交叉淡入淡出过渡；RoomController 动态管理敌人波次状态与自动锁门逻辑。
  • 协程驱动的 Buff 系统：利用 Dictionary<BuffType, Coroutine> 管理动态 Buff 计时器，支持多类型增益（生命/移速/伤害/资源消耗减免）的平滑应用与自动回滚。
Unity C# 2D Roguelite Spine PCG

Global GameJam 2026 作品（占位）

3D 益智解谜游戏，核心系统包括：

  • 空间绘图交互：将传统 UI 绘图转化为 3D 空间内的物理位移交互。玩家需在场景中寻找并按逻辑激活，模拟在场景中绘图的感官体验。开发了序列校验状态机，进行实时路径校验；配合分帧材质动画系统，随交互进程动态映射纹理演变，提供即时的“色彩重现”视觉反馈。
  • 平滑的3D 3C系统设计：实现了具备指令预缓冲逻辑的网格化移动系统，支持指令链式触发，确保操作的连贯性；同步构建了带滞后效果的平滑跟随镜头，结合电影化视角切换，提升整体操作感与演出效果。
  • 室内桌面场景搭建与环境叙事：以“书桌”缩微世界为视角核心，运用Low-poly 风格资源构建微粒化的室内关卡，通过合理的文具与日常物件布局引导探索路径，将游戏主题与环境结构深度融合，实现高沉浸的视觉叙事。
  • 解耦式事件驱动架构：建立全局 TriggerBus，解耦了玩家控制、环境交互与任务进度逻辑。通过观察者模式实现复杂场景下多个“点位”触发器与全局 GameState 的低耦合协作，提升 GameJam 高质量迭代的稳定性。
Unity C# GameJam
